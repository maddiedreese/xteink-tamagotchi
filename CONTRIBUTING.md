# Contributing to XTeInk Tamagotchi

Thank you for your interest in contributing! This project welcomes improvements of all kinds.

## How to Contribute

### 1. Reporting Issues
- Use [GitHub Issues](https://github.com/maddiedreese/xteink-tamagotchi/issues) to report bugs
- Include steps to reproduce, expected behavior, and actual results
- For display issues, include your XTeInk firmware version and device

### 2. Feature Requests
- Open a GitHub Discussion or Issue with the "enhancement" tag
- Describe the use case and how it would help users
- Consider whether it fits the project's scope (simple, skill-based display)

### 3. Pull Requests
- Fork the repository
- Create a feature branch: `git checkout -b my-feature`
- Make your changes
- Test if possible
- Submit a PR with a clear description

### 4. Documentation
- Improvements to README, comments, or guides are always welcome
- Fix typos, clarify steps, add examples

## Code Style

- **Shell scripts:** Use POSIX-compliant syntax (no bash-isms in script.sh)
- **C++ firmware:** Follow Arduino/ESP32 conventions
- **Python:** PEP 8 style

## Questions?

- Start a [GitHub Discussion](https://github.com/maddiedreese/xteink-tamagotchi/discussions)
- Check existing issues first
